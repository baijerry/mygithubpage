Door Alarm System

The Door Alarm System was a hardware project done to learn more about the Arduino, and implemented multiple interrupts, and a custom millis() function. The Arduino control program acts as a security system, which senses when a wooden door is opened. Upon opening, a user must input a password button combination to prevent an alarm from going off. A servo acted as the door's locking latch.

Tools Used

Software: C++

Hardware: Arduino Uno, servo, buttons

Story

To open the servo latch, the user clicks the white button. Upon opening the door, the user then has 10 seconds to input the correct 4 color combination of buttons before the alarm goes off. If the user correctly enters the combination, the system is deactivated. To reactivate the system, the blue button is pressed. Interrupts were used to determine when the white and blue buttons were pressed, and a custom millis function provided the timing mechanism.
