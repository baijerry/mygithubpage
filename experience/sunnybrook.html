Software Developer - Medical Imaging

Sunnybrook Research Institute

What I did

I worked on one major project during the term, as the only co-op on the mammography imaging team. This team of researchers focused on breast cancer imaging and treatment. Their research provided techniques for more accurate detection of breast cancer, included using image processing and data analysis to improve lesion recognition in tissue with non-uniform density.

My project was to develop a radiography imaging system, by integrating an old $60k analog x-ray faxitron with a new $100k digital detector. The two machines had been welded together with an untested DLP-IO20 hardware interface on the faxitron. This project involved both hardware testing and debugging, as well software development to create the control program.

I created a multithreaded C++ control program with a Qt GUI using Boost libraries, custom image manipulation algorithms, and implemented medical DICOM protocols. The resulting program was capable of taking, displaying, and storing x-ray images with proper DICOM metadata. It also provided standard x-ray image processing capabilities including window and levelling, image stacking, gain map generation, and flat-fielding. This program communicated with the hardware system through an Ethernet and COM port.

For the hardware integration, I troubleshot physical system errors by reviewing circuit diagrams, soldered together a new RS232 interface cord, and checked signal quality with various lab tools.

Favourite Moment

When I finished the MVP, I tested this early version of the control program by taking x-ray images of various objects around the room. It was a lot of fun.
